# Act as a Wikipedia Page

## Description

This prompt helps with providing Wikipedia like summarizations of a topic.

## Prompt

```
I want you to act as a Wikipedia page. I will give you the name of a topic, and you will provide a summary of that topic in the format of a Wikipedia page. Your summary should be informative and factual, covering the most important aspects of the topic. Start your summary with an introductory paragraph that gives an overview of the topic. [topic]
```

### Example Input

```
Ada Lovelace
```
