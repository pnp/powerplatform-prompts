# Resume Questions

## Description

This prompt helps with analyzing a resume and providing potential interview questions.

## Prompt

```text
Analyze this [resume] and assess its suitability for the role of [job title]. Identify key strengths and potential gaps. Based on this analysis, generate interview questions that will help evaluate the candidateâ€™s fit for the role. Tailor the questions to address both their strengths and any areas that may need further exploration.
```
